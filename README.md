# 婚礼礼金记录软件

一个免安装、可在 Windows 上直接运行的婚礼礼金记录工具。  
用于记录姓名和金额，动态生成金额排行榜，并支持导出与图片定格留存。

## 功能需求

- 记录内容：姓名、金额
- 排行规则：按金额动态排序并实时显示排行榜
- 重名校验：录入时若发现重名，提示用户并要求重新输入
- 后台实时生成文件：
  - Excel 表格文件
  - Markdown 表格文件
- 一键生成永久定格图片：
  - 图片风格喜庆
  - 图片包含婚礼信息：新郎、新娘、时间、地点
  - 婚礼信息以水印形式叠加在图片中

## 设计与运行要求

- 免安装
- 可在 Windows 上运行
- 界面扁平化设计
- 设计语言参考苹果系统风格
